# Credits

## Original project

GLemples is a browser/WebGL port of **Drempels**, the real-time image-warping
desktop visualiser created by **Ryan Geiss**.

- Author: Ryan Geiss
- Project: Drempels — <https://www.geisswerks.com/about_drempels.html>
- Website: <https://www.geisswerks.com/>

The seven-mode UV warp at the heart of this project (`src/engine/warp.ts`) is a
direct port of Drempels' original warp algorithm, and the "Geiss" mode names and
behaviour are preserved from that work. All credit for the core visual effect and
its mathematics belongs to Ryan Geiss.

Drempels' source was released under the GNU General Public License, which is why
this port is likewise distributed under the GPL (see [LICENSE](LICENSE)).
